最小数字

源文件：min.bas
输入文件：min.in
输出文件：min.out

在n个数字中找出其中最小的数字。

输入：
第一行为一个正整数，表示数字的个数n(1<=n<=100);
第二行为使用空格分隔的n个数字，每个数字f都满足：0<=f<=10000。

输出：
输出这n个数字中最小的数字的序号和最小数字，用空格分隔。

【样例输入1】
3
2 1 5

【样例输出1】
2 1

【说明】
2 1 5三个数字中最小数字为1，它在这三个数字中排位第二，所以序号为2。

【样例输入2】
3
0 0 0

【样例输出2】
1 0

【说明】
0 0 0三个数字中最小数字为0，但是同时出现三个最小数字，这时序号取较小的那个，即1。
